What is "lstsq" for PCA template?
    "least squares"
    Minimizes residual ||Ax - b||^2

    PCA: Get a set of eigenvectors to form a basis
    least squares function (NumPy/SciPy - which one does Pyqsofit) is an alternative for a dot product for representing data point in terms of templates
    A x = b
    A is PCA template matrix (comprised of eigenvectors/principal components)
    b is data vector you want to decompose
    x is coefficients you are solving for

    PCA: Principal Component Analysis
    - PCA templates come from a library of established spectra
    - Templates form a basis that can be linearly combined
    - Combining PCA models reconstructs a host galaxy model
    - "explained variance plot" - do we have those? can I make them? should I make them?
        - scree plot - plotting explained variance vs. number of components
    - I am going to assume that the library PyQSOFit uses is the SDSS galaxy spectra
    - Need to know the general class of the galaxy before running with PCA
        - Ask Noah for the classes of objects later and re-run bulk fitting with PCA template?
    - Able to directly choose the number of components (dimensions) that the PCA template includes, able to filter the spectra chosen to create a better model
        - For AGN, standard is 5-10, captures ~95% of variance w/out fitting noise
